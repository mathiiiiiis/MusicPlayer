### Prerequisites

* Java 19 or higher
* JavaFX SDK

### Setup

1. Create `.env`-file:

```bash
JAVAFX_PATH=/path/to/javafx-sdk/lib/*
```

2. Compile and run using:

```bash
make run
```

3. To clean compiled files:

```bash
make clean
```
